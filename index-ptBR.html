Este é um clone da plataforma Reddit focado na sua funcionalidade do backend.
O projeto consiste do frontend em Next, utilizando URQL para realizar SSR (server-side rendering) em algumas páginas, assim como cache das requisições. Para uma melhor experiência de desenvolvimento, foi utilizado a ferramenta graphql-codegen para gerar hooks prontos para utilização das queries e mutations, enquanto para o UI/UX, foi usado Chakra UI.
No servidor, foi construida uma api com GraphQL com TypeGraphQL para a criação de schemas, Apollo Server para fazer a requisição dos dados, em conjunto com Redis para armazenar os cookies da sessão, nodemailer para mandar emails automaticamente, TypeORM para criação de queries e interações em geral com o banco de dados, finalizando com dataloader para fazer batching e cache das requisições.
